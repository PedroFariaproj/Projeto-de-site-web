# 📝 Projeto Lista de Tarefas (To-Do List)

Este é um projeto simples de **Lista de Tarefas (To-Do List)** desenvolvido utilizando **HTML, CSS e JavaScript**.
O objetivo do projeto é permitir que o usuário adicione, visualize, marque como concluídas e remova tarefas de forma prática diretamente no navegador.

##  Funcionalidades

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Remover tarefas da lista
* Visualizar todas as tarefas cadastradas

##  Tecnologias Utilizadas

* **HTML5** – Estrutura da página
* **CSS3** – Estilização da interface
* **JavaScript (Vanilla JS)** – Lógica e interatividade

##  Estrutura do Projeto

```
/projeto-tarefas
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ Como Executar o Projeto

1. Faça o download ou clone este repositório:

```
git clone https://github.com/seu-usuario/projeto-tarefas.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo **index.html** em qualquer navegador.
